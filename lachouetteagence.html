La mission

Contexte

Travaillant chez La Chouette Agence. C'est une grande agence de web design basée à Lyon. L'activité de l'entreprise est en perte de vitesse. L'entreprise n'apparait qu'à la deuxième page du moteur de recherche si on entre “Entreprise web design Lyon”.

Consignes

La Chouette Agence cherche avant tout à améliorer son référencement. , tout en suivant certaines consignes :

  • Analyser l’état actuel de SEO du site fourni
    • Indiquer les parties du site qui ne sont pas optimisées pour le SEO et/ou qui ne sont pas à jour en termes d’accessibilité ;
    • Justifier les choix ;
    • Ajouter la bonne pratique à mettre en place pour chaque partie du site ;
    • Citer les sources ;
    • Sélectionner 10 recommandations pour améliorer le site
    • Faire au moins une recommandation sur la vitesse et la taille du site et une sur l’accessibilité ;
    • Ajouter les éléments nécessaires.
  • Amélioration du SEO du site
    • Optimiser le site en appliquant les 10 recommandations à son contenu et à son code source ;
    • Fournir le code source complet de la version améliorée du site ;
    • Vérifier le code HTML et le CSS sur W3C.
