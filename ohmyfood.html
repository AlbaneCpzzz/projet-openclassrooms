Ohmyfood

Dynamisez une page web avec des animations CSS

La mission

Contexte

Recrutée en tant que développeur junior chez Ohmyfood. C'est une entreprise de commande de repas en ligne. Leur site permet aux usagés de composer leur propre menu et réduire leur temps d’attente dans les restaurants car leur menu est préparé à l’avance.

Consignes

Ohmyfood souhaite avant tout développer un site proposant le menu de 4 grands restaurants parisiens et à permettre la réservation en ligne et la composition de menus des utilisateurs. Pour ce faire, ma mission est de dynamiser les pages web avec des animations CSS , tout en suivant certaines consignes :

  • Animer les boutons avec un éclaircissement et une intensification de l'ombre ;
  • Créer un bouton "J'aime" en forme de coeur qui se remplit progressivement au survol ;
  • Renvoyer les clics sur le bouton “Contact” au mail de la société ;
  • Créer un “loading spinner” de 1 à 3 secondes quand on arrive sur la page d'accueil qui couvrira l'intégralité de l'écran ;
  • Faire apparaitre progressivement les plats avec un léger décalage dans le temps ;
  • Faire apparaitre une petite coche au survol des plats.

Contraintes

Le projet doit suivre les contraintes techniques ci-dessous :

  • Développement sur CSS et HTML, avec l’utilisation de SASS et sans JavaScript ;
  • Versionnage sur GitHub ;
  • Création de trois versions : desktop (libre), mobile (maquette) et tablette (libre) ;
  • Vérification faite avec W3C HTML et CSS ;
  • Le site doit être parfaitement compatible avec les dernières versions desktop de Chrome et Firefox ;
  • Couleurs de la charte : Primaire #9356DC / Secondaire #FF79DA / Tertiaire #99E2D0 ;
  • Police : Shrikhand & Roboto ;

Le résultat

Le projet

Le projet a été réalisé sur l'éditeur de code, Visual Studio Code, versionnné sur GitHub et hébergé sur GitHub Pages.

W3C

La vérification W3C a été effectuée : les codes CSS et HTML ont été validés.
