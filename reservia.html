La mission

Contexte

Stage en tant que développeur web chez Reservia. C'est une entreprise proposant un outil de planification de vacances. Leur site permet aux usagers de trouver des hébergements et des activités dans la ville de leur choix. Les hébergements peuvent également être filtrés par thématique. L’entreprise décide de réaliser un prototype.

Consignes

Ma mission consiste à intégrer la maquette responsive en HTML et CSS , tout en suivant certaines consignes :

  • Intégrer un champ de recherche pour chercher des hébergements dont le texte peut être édité par l’usager ;
  • Rendre cliquable toutes les cartes dans leur intégralité ;
  • Animer les filtres au survol ;
  • Créer des liens sur la barre de navigation pour qu'ils renvoient aux sections de la page.

Contraintes

Le projet doit suivre les contraintes techniques ci-dessous :

  • Bibliothèque pour les icones : Font Awesome
  • Couleurs de la charte : Bleu #0065FC / Bleu claire #DEEBFF / Gris pour le fond #F2F2F2
  • Police : Raleway
  • Création de trois versions : desktop (maquette), mobile (maquette) et tablette (libre)
  • Vérification faite avec W3C HTML et CSS
  • Utilisation de Visual Studio Code

Le résultat

Le projet

Le projet a été réalisé sur l'éditeur de code, Visual Studio Code, versionnné sur GitHub et hébergé sur GitHub Pages.

W3C

La vérification W3C a été effectuée : les codes CSS et HTML ont été validés.
